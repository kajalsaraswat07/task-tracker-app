# Task Tracker App

A simple and user-friendly Task Tracker web application built using React.  
This application allows users to add tasks, view all tasks, mark tasks as completed, edit tasks, and delete tasks.  
All tasks are stored in the browser using localStorage, so data persists even after page refresh.

---

## Features

- Add new tasks
- View all tasks in a list
- Mark tasks as **Pending** or **Completed**
- Edit existing tasks
- Delete tasks
- Data persistence using **localStorage**
- Clean and responsive UI

---

## Tech Stack

- React (Vite)
- JavaScript
- HTML
- CSS
